核心理念

vim 的使用是遵循语法的
动词：
i insert，c change，r replace，d delete，y yank，p paste，f find，v visual，u undo，a append
j join
x 删除(因为以前的打字机如果要删除文字，是用xxxx覆盖那个文字)
介词：i in，a around，t to，f forward
名词：w word，p paragraph，t tag，s sentence
举例子：
  1. 改大小写fcrC
    find c replace to C 找到小写c替代为大写C
  2. 改函数内容ci{
    change in {}改变括号{}里的内容
  3. 移动文本位置dwf.p
    delete word find . paste 删除单词，寻找符号 . ，粘贴至符号后
  4. 删除粘贴操作d$
    delete to the end of the line删除至行末
