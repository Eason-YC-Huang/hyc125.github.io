剑指 Offer 04. 二维数组中的查找
如下图所示，我们将矩阵逆时针旋转 45° ，并将其转化为图形式，发现其类似于 二叉搜索树 ，即对于每个元素，其左分支元素更小、右分支元素更大。因此，通过从 “根节点” 开始搜索，遇到比 target 大的元素就向左，反之向右，即可找到目标值 target 。
public boolean findNumberIn2DArray(int[][] matrix, int target) {
    if (matrix.length == 0) return false;
    int m = 0, n = matrix[0].length - 1;
    while (m < matrix.length && n >= 0) {
        int num = matrix[m][n];
        if (num > target) n--;
        else if (num < target) m++;
        else return true;
    }
    return false;
}
复杂度分析： 时间复杂度 O(M+N)O(M+N)O(M+N) ：其中，N 和 M 分别为矩阵行数和列数，此算法最多循环 M+NM+NM+N次。 空间复杂度 O(1) : i, j 指针使用常数大小额外空间。
剑指 Offer 11. 旋转数组的最小数字
public int minArray(int[] nums) {
    int len = nums.length;
    int lo=0 , hi=len-1;
    while(hi>=0&&nums[lo]==nums[hi])hi--;
    if(hi<0) return nums[lo];
    int target = nums[hi];
    while(lo<hi){
        int mid = lo+((hi-lo)>>1);
        if(nums[mid]>target) lo = mid + 1;
        else hi = mid;
    }
    return nums[lo];
}

剑指 Offer 21. 调整数组顺序使奇数位于偶数前面
左右指针，左指针从左往右找寻偶数，右指针从右往左找寻奇数，然后调换位置
// 左右指针
public int[] exchange(int[] nums) {
    int len = nums.length;
    int lo = 0 , hi = len - 1;
    while(true){
        while(lo<len&&nums[lo]%2==1) lo++;
        while(hi>=0&&nums[hi]%2==0) hi--;
        if(lo>=hi) break;
        swap(nums,lo,hi);
    }
    return nums;
}

private void swap(int[]nums,  int i , int j){
    int t = nums[i];
    nums[i] = nums[j];
    nums[j] = t;
}
剑指 Offer 29. 顺时针打印矩阵
整个打印过程就是从左往右，从上往下，从右往左，从下往上
一个二维矩阵拥有上下左右边界
从左往右结束，上指针+1
从上往下结束，右指针-1
从右往左结束，下指针-1
从下往上结束，左指针+1
public int[] spiralOrder(int[][] matrix) {
    if (matrix.length == 0) return new int[0];
    int m = matrix.length, n = matrix[0].length;
    int[] ans = new int[m * n];
    int U = 0, D = m - 1, L = 0, R = n - 1, x = 0;
    while (true) {
        for (int i = L; i <= R; i++) ans[x++] = matrix[U][i];
        if (++U > D) break;
        for (int i = U; i <= D; i++) ans[x++] = matrix[i][R];
        if (--R < L) break;
        for (int i = R; i >= L; i--) ans[x++] = matrix[D][i];
        if (--D < U) break;
        for (int i = D; i >= U; i--) ans[x++] = matrix[i][L];
        if (++L > R) break;
    }
    return ans;
}

剑指 Offer 39. 数组中出现次数超过一半的数字
本题常见的三种解法：
  • 哈希表统计法： 遍历数组 nums ，用 HashMap 统计各数字的数量，即可找出众数 。此方法时间和空间复杂度均为 O(N)。
  • 数组排序法： 将数组 nums 排序，数组中点的元素一定为众数。
  • 摩尔投票法： 核心理念为票数正负抵消 。此方法时间和空间复杂度分别为O(N) 和O(1)。
public int majorityElement(int[] nums) {
    int len = nums.length;
    int ans=0, cnt=0;
    for(int i=0;i<len;i++){
        if(cnt==0){
            ans=nums[i];
            cnt++;
        }
        else if(ans==nums[i]) cnt++;
        else cnt--;
    }
    return ans;
}

public int search(int[] nums, int target) {
    // 二分找出该元素第一次出现的位置然后向后统计计数
    if(nums.length==0)return 0;
    int len = nums.length;
    int lo = 0 , hi = len-1;
    while(lo<hi){
        int mid = lo+((hi-lo)>>1);
        if(nums[mid]>=target) hi=mid;
        else lo=mid+1;
    }
    if(nums[lo]!=target) return 0;
    int ans = 1;
    while(++lo<len&&nums[lo]==target) ans++;
    return ans;
}

亮灯泡💡
100个灯泡组成一个圆环，按一下灯泡会修改该灯泡及其左右灯泡的暗亮状态。
如何让所有灯泡处于亮的状态？
假设只有1个亮，剩余99个都是暗的，剩余99个，3个3个一组，按下中间的灯泡，这样所有的灯泡都亮了
怎么样可以变成1个亮，99个暗？
一遇到亮的就摸它后一个灯泡让它变暗，但是最后一个灯泡不能摸，因为会修改第一个灯泡。也就是可以让1~98的灯泡的都变暗。
最后两个灯泡，有三种情况：
  • 一暗一亮，到达目的
  • 两个亮，摸一下最后一个灯泡，除了第一个灯泡，其它灯泡都变暗了，达到目的
  • 两个暗，也就是全部都暗了，每个灯泡摸一下，每个灯泡都会发送三次变换，也就是暗的会变成亮的，所以只需要把所有灯泡摸一下即可
