什么是二叉树？

二叉树是每个结点最多有两个子树的树结构。子树通常被称作“左子树”和“右子树”。

术语

树的高度：描述的是根节点到叶节点的最长距离。叶节点的高度为1.
节点的高度 : 描述的是节点到叶节点的最长距离。叶节点的高度为1.
节点的深度：描述的是节点到根节点的最长距离。根节点的深度为1.
满二叉树：除了叶结点外每一个结点都有左右子叶且叶子结点都处在最底层的二叉树。
完全二叉树：设二叉树的高度为h，除第 h 层外，其它各层的结点数都达到最大个数，第h层有叶子结点，并且叶子结点都是从左到右依次排布，这就是完全二叉树。
平衡二叉树：一棵空树或它左右两个子树都是一棵平衡二叉树，且左右子树的高度差不超过1。

public List<Integer> preOrder(TreeNode root) {
    List<Integer> ans = new ArrayList<>();
    if (root == null) return ans;
    Deque<TreeNode> stack = new ArrayDeque<>();
    stack.push(root);
    while (!stack.isEmpty()) {
        TreeNode cur = stack.pop();
        ans.add(cur.val);
        if (cur.right != null) stack.push(cur.right);
        if (cur.left != null) stack.push(cur.left);
    }
    return ans;
}


public List<Integer> preOrder(TreeNode root) {
  List<Integer> ans = new LinkedList<Integer>();
  if(root == null) return ans;
  ans.add(root.val);
  ans.addAll(preOrder(root.left));
  ans.addAll(preOrder(root.right));
  return ans;
}

public List<Integer> inOrder(TreeNode root) {
    List<Integer> ans = new ArrayList<>();
    if (root == null) return ans;
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode cur = root;
    while (cur != null || !stack.isEmpty()) {
        while (cur != null) {
            stack.push(cur);
            cur = cur.left;
        }

        if (!stack.isEmpty()) {
            cur = stack.pop();
            ans.add(cur.val);
            cur = cur.right;
        }
    }
    return ans;
}

public void iterate(TreeNode node) {
    // 模拟调用栈
    Deque<StackFrame<TreeNode>> stack = new ArrayDeque<>();
    stack.push(new StackFrame<>(false, node));

    while (!stack.isEmpty()) {
        StackFrame<TreeNode> frame = stack.pollLast();
        TreeNode cur = frame.param;
        if (cur == null) continue;
        if (frame.seen) {
            operation(cur);
        } else {
            stack.offerLast(new StackFrame<>(true, cur));
            stack.offerLast(new StackFrame<>(false, cur.right));
            stack.offerLast(new StackFrame<>(false, cur.left));
        }
    }
}

private TreeNode dfs(int[] post, int is, int ie , int ps, int pe){
    if(ps>pe) return null;
    int val = post[pe];
    int idx = seen.get(val);
    TreeNode node = new TreeNode(val);
    int leftSize = idx-is;
    node.left  = dfs(post,is,   is+leftSize-1,ps,         ps+leftSize-1);
    node.right = dfs(post,idx+1,ie,           ps+leftSize,pe-1);
    return node;
}

private TreeNode dfs(int[] preorder, int pl , int pr , int il , int ir) {
    if(pl>pr) return null;
    int val = preorder[pl];
    int idx = seen.get(val);
    TreeNode node = new TreeNode(val);
    int leftSize = idx-il;
    node.left = dfs(preorder,pl+1,pl+leftSize,il,idx-1);
    node.right = dfs(preorder,pl+leftSize+1,pr,idx+1,ir);
    return node;
}
297. 二叉树的序列化与反序列化
使用打印空节点的前序遍历方式进行序列化。
因为二叉树前序遍历的输出具有[根节点，左子树，右子树]的特性。
也就是对于一个二叉树前序遍历输出的列表，我们用列表的第一个元素构造当前节点，然后用列表剩余元素去构造该节点的左子树和右子树。
例如[1,2,3] 输出空节点的前序遍历 [1,2,null,null,3,null,null]
public class Codec {

    private static final String NULL = "null";

    public String serialize(TreeNode root) {
        StringBuilder sb = new StringBuilder();
        doSerialize(root,sb);
        return sb.toString();
    }

    private void doSerialize(TreeNode node,StringBuilder sb){
        if(node==null){
            sb.append(NULL).append(",");
            return;
        }
        sb.append(node.val).append(",");
        doSerialize(node.left,sb);
        doSerialize(node.right,sb);
    }

    public TreeNode deserialize(String data) {
        // 原来split后的得到的数组，最后一个空字符串会被去掉
        // 1,2,3, -> [1,2,3]
        String[] dataArr = data.split(",");
        LinkedList<String> dataList = new LinkedList<>(Arrays.asList(dataArr));
        return doDeserialize(dataList);
    }

    private TreeNode doDeserialize(LinkedList<String> dataList){
        String val = dataList.removeFirst();
        if(NULL.equals(val)){
            return null;
        }

        TreeNode node = new TreeNode(Integer.parseInt(val));
        node.left = doDeserialize(dataList);
        node.right = doDeserialize(dataList);
        return node;
    }

}

public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> ans = new ArrayList<>();
    if (root == null) return ans;
    
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    while (!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> sub = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode cur = queue.poll();
            if (cur.left != null) queue.offer(cur.left);
            if (cur.right != null) queue.offer(cur.right);
            sub.add(cur.val);
        }
        ans.add(sub);
    }
    return ans;
}

  if(root == null) return 0;
  int left = minDepth(root.left);
  int right = minDepth(root.right);
  return (left == 0 || right == 0) ? left + right + 1: Math.min(left,right) + 1;

private static class IdNode {
    int id;
    TreeNode node;

    public IdNode(int id, TreeNode node) {
        this.id = id;
        this.node = node;
    }
}

public int widthOfBinaryTree(TreeNode root) {
    int ans = 0;
    if (root == null) return ans;
    Deque<IdNode> queue = new LinkedList<>();
    queue.offer(new IdNode(0, root));
    while (!queue.isEmpty()) {
        int size = queue.size();
        int width = queue.peekLast().id - queue.peekFirst().id + 1;
        ans = Math.max(ans, width);
        for (int i = 0; i < size; i++) {
            IdNode idNode = queue.poll();
            if (idNode.node.left != null) queue.offer(new IdNode(2 * idNode.id + 1, idNode.node.left));
            if (idNode.node.right != null) queue.offer(new IdNode(2 * idNode.id + 2, idNode.node.right));
        }
    }
    return ans;
}

public List<List<Integer>> pathSum(TreeNode node, int targetSum) {
    if (node == null) return ans;
    targetSum -= node.val;
    sub.add(node.val);
    if (node.left == null && node.right == null && targetSum == 0) ans.add(new ArrayList<>(sub));
    pathSum(node.left, targetSum);
    pathSum(node.right, targetSum);
    sub.remove(sub.size() - 1);
    return ans;
}

560. 和为 K 的子数组
先求出前缀和，然后朴素的做法是遍历到第i个位置的时候，再去遍历[0,j](j<i)位置，如果sum[i]-k==sum[j]，结果数加1。
public int subarraySum(int[] nums, int k) {
    int ans = 0;
    int len = nums.length;
    int[] sum = new int[len + 1];
    for (int i = 1; i <= len; i++)
        sum[i] = sum[i - 1] + nums[i - 1];

    for (int i = 1; i <= len; i++) {
        for (int j = 0; j < i; j++) {
            if (sum[i] - k == sum[j])
                ans++;
        }
    }

    return ans;
}
时间复杂度是O(N2)O(N^{2})O(N2)，空间复杂度O(N)O(N)O(N)
仔细观察可以发现，遍历j只是为了统计在sum[0,j]里，有多少个数是等于sum[i] - k的，我们可以利用一个哈希表加速该过程。
public int subarraySum(int[] nums, int k) {
    int ans = 0;
    int len = nums.length;
    int[] sum = new int[len + 1];
    for (int i = 1; i <= len; i++)
        sum[i] = sum[i - 1] + nums[i - 1];

    // Map<前缀和,出现的次数>
    Map<Integer, Integer> map = new HashMap<>();
    map.put(0, 1);

    for (int i = 1; i <= len; i++) {
        int count = map.getOrDefault((sum[i] - k), 0);
        ans += count;
        map.put(sum[i], map.getOrDefault(sum[i], 0) + 1);
    }

    return ans;
}

236. 二叉树的最近公共祖先
  • 如果node为null，返回null，也就是返回node。
  • 如果node==p或者node==q，说明node就是p和q的最近公共祖先，返回node。
  • 判断node的左右子树是否都能找到p或者q，如果是，则node是p和q最近公共祖先。
  • 否则就从找到p或者q的那颗子树开始查找p和q的最近公共祖先。
public TreeNode lowestCommonAncestor(TreeNode node, TreeNode p, TreeNode q) {
    if (node == null || node == p || node == q) return node;
    boolean left = dfs(node.left, p, q);
    boolean right = dfs(node.right, p, q);
    // 左右子树都能找到p或者q，说明当前节点就是p和q的最近祖先节点
    if (left && right) return node;
    // p和q的最近祖先节点在node的左节点
    if (left) return lowestCommonAncestor(node.left, p, q);
    // p和q的最近祖先节点在node的右节点
    if (right) return lowestCommonAncestor(node.right, p, q);
    return null;
}

// 能否从node开始找到p或者q
private boolean dfs(TreeNode node, TreeNode p, TreeNode q) {
    if (node == null) return false;
    if (node == p || node == q) return true;
    return dfs(node.left, p, q) || dfs(node.right, p, q);
}
仔细观察lowestCommonAncestor和dfs方法，你会发现它们之间的功能有重叠。
lowestCommonAncestor：从node开始，找出p和q的最近公共祖先。
dfs：判断node是否有p或者q。
如果能从node开始找出p和q的最近公共祖先，那么node必然包含p和q。
也就是可以通过lowestCommonAncestor方法代替dfs方法。
如果lowestCommonAncestor方法返回非null，说明能找到p或者q。
public TreeNode lowestCommonAncestor(TreeNode node, TreeNode p, TreeNode q) {
    if (node == null || node == p || node == q) return node;
    TreeNode left = lowestCommonAncestor(node.left, p, q);
    TreeNode right = lowestCommonAncestor(node.right, p, q);
    if(left==null) return right;
    if(right==null) return left;
    return node;
}

public int minDiffInBST(TreeNode node) {
    if(node==null) return ans;
    minDiffInBST(node.left);
    if(prev != null) ans = Math.min(ans,Math.abs(node.val-prev.val));
    prev = node;
    minDiffInBST(node.right);
    return ans;
}
