Trie树，又称单词查找树或字典树，主要用来存储和查询字符串，利用字符串的公共前缀来减少查询时间，最大限度地减少无谓的字符串比较。例如，下图是存储了hello hi her how see so 5个单词的Trie树。
每个红色的点意味着从它到根节点的路径是一个单词。
前缀树模板
private static class Node {
    boolean isWord = false;
    Map<Character, Node> child = new HashMap<>();
}

private Node dummy = new Node();

public void add(String str) {
    Node cur = dummy;
    for (int i = 0; i < str.length(); i++)
        cur = cur.child.computeIfAbsent(str.charAt(i), x -> new Node());
    cur.isWord = true;
}

public boolean contains(String str) {
    Node cur = dummy;
    for (int i = 0; i < str.length(); i++) {
        Node child = cur.child.get(str.charAt(i));
        if (child==null) return false;
        cur = child;
    }
    return cur.isWord;
}


472. 连接词
1.字符串按照长度从短到长排序，确保当前处理的字符，其组合字符已经记录在前缀树
2.当前字符匹配到一个组合字符时，有两种情况，开始新字符的匹配，继续匹配
private static class Node {
    boolean isWord = false;
    Map<Character, Node> child = new HashMap<>();
}

private Node dummy = new Node();

public void add(String str) {
    Node cur = dummy;
    for (int i = 0; i < str.length(); i++)
        cur = cur.child.computeIfAbsent(str.charAt(i), x -> new Node());
    cur.isWord = true;
}

public boolean contains(String str) {
    Node cur = dummy;
    for (int i = 0; i < str.length(); i++) {
        Node child = cur.child.get(str.charAt(i));
        if (child == null) return false;
        cur = child;
    }
    return cur.isWord;
}

public List<String> findAllConcatenatedWordsInADict(String[] words) {
    // 看到题目第一反应就是前缀树,但是在思考[cat,cats,dog,catsdog]尬住了
    // ⭐️⭐️⭐️当你遇到这样可能可以,那样也可能可以的情况时,要考虑下dfs⭐️⭐️⭐️
    List<String> ans = new ArrayList<>();

    Arrays.sort(words, Comparator.comparingInt(String::length));
    for (String word : words) {
        if (word.length() == 0) continue;
        if (dfs(word, 0)) ans.add(word);
        else add(word);
    }

    return ans;
}

// str[idx:]是否是连接词
private boolean dfs(String str, int idx) {
    if (idx == str.length()) return true;
    Node cur = dummy;
    for (int i = idx; i < str.length(); i++) {
        Node child = cur.child.get(str.charAt(i));
        if (child == null) return false;
        if (child.isWord && dfs(str, i + 1)) return true;
        else cur = child;
    }
    return false;
}
