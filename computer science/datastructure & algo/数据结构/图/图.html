图的基本概念

顶点(vertex)：图中元素叫作顶点
边(edge)：连接两个顶点的叫作边
平行边：两条或以上的边都连向同样的两个顶点，这些边称为平行边
自环边：边的两段连向的是同一个顶点
复杂图：拥有自环边或平行边的图称为复杂图
有向图和无向图：如果边具有方向性则为有向图、否则是无向图
度(degree)：表示顶点与多少条边相连。在有向图中度分为入度(in-degree)和出度(out-degree)
degree叫度怪怪的，为什么不叫数量，它明明也有数量的意思，amount or level of something
权重：边所携带的信息
根据以上信息，我们可以整理出图具有以下分类

图的存储方式

邻接矩阵

图最直观的一种存储方式是邻接矩阵(adjacency matrix)。邻接矩阵底层依赖二维数据。
对于无向图，如果顶点i与顶点j之间有边，我们就将A[i][j]和A[j][j]标记为1；对于有向图，如果有一条从顶点i指向顶点j的边，我们就将A[i][j]标记为1。同理，如果有一条从顶点j指向顶点i的边，我们就将A[j][i]标记为1。对于带权图，数组中存储相应的权重。
用邻接矩阵来表示图，对应的代码如下表示。
public class Graph{

    private int V;
    private boolean matrix[][];
    
    public Graph(int V){
        this.V = V;
        matrix = new boolean[V][V];
    }

    public void addEdge(int i, int j) {
        matrix[i][j] = true;
        matrix[j][i] = true;
    }
}
因为邻接矩阵底层依赖数组，所以，在邻接矩阵中获取两个顶点之间关系的操作很简单。同时，这种存储方式还有一个好处，那就是可以将图中的很多运算转换成矩阵运算。
对于稀疏图，顶点很多，但是每个顶点的边并不多，如果用邻接矩阵来存储，会非常浪费存储空间

邻接表

针对邻接矩阵比较浪费内存空间的问题，我们来看另一种图的存储方法：邻接表(adjacency list)。
邻接表为每个节点分配了一条链表，用来记录该节点与哪些节点相连。
class Graph {

    private int V; // 顶点的个数
    private LinkedList<Integer> adj[]; // 邻接表

    public Graph(int v) {
        this.V = v;
        adj = new LinkedList[V];
        for (int i = 0; i < V; i++) {
            adj[i] = new LinkedList<>();
        }
    }

    public void addEdge(int s, int t) {
        adj[s].add(t);
        adj[t].add(s);
    }
}
这种实现方式跟玩似的0.0，应该也是刷题才这么写吧，工业级图的实现可以看下guava的
一般都是使用邻接表来存储图，这种存储方式更加灵活

public void bfs(int start, int end) {
    if (start == end) {
        return;
    }
    // 记录已经被访问的顶点，避免顶点被重复访问
    boolean[] visited = new boolean[v];
    visited[start] = true;
    Queue<Integer> queue = new LinkedList<>();
    queue.add(start);
    // prev[] 记录搜索路径
    // prev[next] = cur 存储的是顶点next的前驱顶点
    // 当从顶点start搜索到顶点end后
    // prev数组中存储从顶点start到顶点end所经历的路径
    // 不过，这条路径是反向存储的
    // end -> prev[end] -> prev[prev[end]] -> ... -> start
    // 所以在获取路径时，我们需要递归来输出正向路径
    int[] prev = new int[v];
    Arrays.fill(prev, -1);
    while (queue.size() != 0) {
        int cur = queue.poll();
        for (Integer next : adj[cur]) {
            if (visited[next]) {
                continue;
            }
            prev[next] = cur;
            if (next == end) {
                print(prev, start, end);
                return;
            }
            visited[next] = true;
            queue.offer(next);
        }
    }
}

private void print(int[] prev, int start, int end) {
    if (end != start) {
        print(prev, start, prev[end]);
    }
    System.err.println(end + " ");
}

空间复杂度O(V)O(V)O(V)

public void dfs(int start, int end) {
    found = false;
    boolean[] visisted = new boolean[v];
    int[] prev = new int[v];
    Arrays.fill(prev, -1);
    doDfs(start, end, visisted, prev);
    print(prev, start, end);
}

private void doDfs(int cur, int end, boolean[] visited, int[] prev) {
    if (found) return;
    visited[cur] = true;
    if (cur == end) {
        found = true;
        return;
    }

    for (Integer next : adj[cur]) {
        if (!visited[next]) {
            prev[next] = cur;
            doDfs(next, end, visited, prev);
        }
    }
}

Topological Sorting
// B被A依赖,则在B的链表上加上A
private List<Integer>[] graph;

public void topologicalSort() {
    // 统计每个节点的入度(它依赖多少个节点)
    int[] cnt = new int[graph.length];
    for (List<Integer> nodeList : graph) {
        for (Integer node : nodeList) {
            cnt[node]++;
        }
    }
    // 找出没有依赖的节点,他们可以先执行
    Deque<Integer> queue = new ArrayDeque<>();
    for (int i = 0; i < cnt.length; i++) {
        if (cnt[i] == 0) queue.add(i);
    }
    // 执行节点
    while (!queue.isEmpty()) {
        Integer node = queue.poll();
        System.out.println(" ->" + node);
        // 这个节点已经被执行了,依赖它的数量可以减一了
        for (Integer rely : graph[node]) {
            cnt[rely]--;
            if (cnt[rely] == 0) queue.offer(rely);
        }
    }
}

public boolean canFinish(int numCourses, int[][] prerequisites) {
    Map<Integer, List<Integer>> graph = new HashMap<>();
    int[] seen = new int[numCourses];
    for (int[] pre : prerequisites) {
        graph.computeIfAbsent(pre[1], key -> new ArrayList<>()).add(pre[0]);
        seen[pre[0]]++;
    }

    Deque<Integer> queue = new ArrayDeque<>(numCourses);
    for (int i = 0; i < seen.length; i++)
        if (seen[i] == 0)
            queue.offer(i);


    while (!queue.isEmpty()) {
        Integer course = queue.poll();
        numCourses--;
        List<Integer> dependence = graph.get(course);
        if (dependence != null) {
            for (Integer c : dependence) {
                seen[c]--;
                if (seen[c] == 0) queue.offer(c);
            }
        }
    }
    return numCourses == 0;
}

public int[] findOrder(int numCourses, int[][] prerequisites) {
    int[] ans = new int[numCourses];
    int idx = 0;
    Map<Integer, List<Integer>> graph = new HashMap<>();
    int[] seen = new int[numCourses];
    for (int[] pre : prerequisites) {
        graph.computeIfAbsent(pre[1], key -> new ArrayList<>()).add(pre[0]);
        seen[pre[0]]++;
    }

    Deque<Integer> queue = new ArrayDeque<>(numCourses);
    for (int i = 0; i < seen.length; i++)
        if (seen[i] == 0)
            queue.offer(i);

    while (!queue.isEmpty()) {
        Integer course = queue.poll();
        ans[idx++] = course;
        List<Integer> dependence = graph.get(course);
        if (dependence != null) {
            for (Integer c : dependence) {
                seen[c]--;
                if (seen[c] == 0) queue.offer(c);
            }
        }
    }
    return idx == numCourses ? ans : new int[0];
}
