用 n 表示股票价格数组的长度；
用 i 表示第 i 天（i 的取值范围是 0 到 n - 1）；
用 k 表示允许的最大交易次数；
用 T[i][k] 表示在第 i 天结束时，最多进行 k 次交易的情况下可以获得的最大收益。

T[i][k][0] 表示在第 i 天结束时，最多进行 k 次交易且在进行操作后持有 0 份股票的情况下可以获得的最大收益；
T[i][k][1] 表示在第 i 天结束时，最多进行 k 次交易且在进行操作后持有 1 份股票的情况下可以获得的最大收益。

对于状态转移方程中的 T[i][k][0]，第 i 天进行的操作只能是休息或卖出，因为在第 i 天结束时持有的股票数量是 0。
对于状态转移方程中的 T[i][k][1]，第 i 天进行的操作只能是休息或买入，因为在第 i 天结束时持有的股票数量是 1。

T[i][k][0] = max(T[i - 1][k][0], T[i - 1][k][1] + prices[i])
T[i][k][1] = max(T[i - 1][k][1], T[i - 1][k - 1][0] - prices[i])

int[][] dp = new int[prices.length][2];
dp[0][0]=0;
dp[0][1]=-prices[0];
for(int i=1;i<prices.length;++i){
  dp[i][0] = Math.max(dp[i-1][0],dp[i-1][1]+prices[i]);
  dp[i][1] = Math.max(dp[i-1][1],-prices[i]);
}
return dp[prices.length-1][0];

// 如果注意到第 i 天的最大收益只和第 i - 1 天的最大收益相关，空间复杂度可以降到 O(1)。
int p0=0;
int p1=-prices[0];
for(int i=1;i<prices.length;++i){
  p0 = Math.max(p0,p1+prices[i]);
  p1 = Math.max(p1,-prices[i]);
}
return p0;
122. 买卖股票的最佳时机 II
public int maxProfit(int[] prices) {
  // dp[i][0] 直至第i天结束且手上没有股票时能得到的最大利润
  // dp[i][1] 直至第i天结束且手上有股票时能得到的最大利润

  // dp[i][0] = max(dp[i-1][0],dp[i-1][1]+prices[i])
  // dp[i][1] = max(dp[i-1][1],dp[i-1][0]-prices[i])
  if(prices==null ||prices.length<2) return 0;

  int p0=0,p1=-prices[0];
  int len = prices.length;
  for(int i=1;i<len;++i){
    p0 = Math.max(p0,p1+prices[i]);
    p1 = Math.max(p1,p0-prices[i]);
  }
  return p0;
}
