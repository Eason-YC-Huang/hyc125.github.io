什么是前缀数组？
前缀数组的每一个元素pre[i]表示一个数列的前i项操作的结果
  • 前缀和数组: sum[i]表示数列前i项求和的结果
  • 前缀异或数组: sum[i]表示数列前i项异或的结果
  • ......
前缀数组的意义？
前缀数组常用于区间求值。例如我们要求解任意一个区间的和、求解有多少个区间的和满足是k的倍数，等等

二维前缀和
int[][] sum;

public NumMatrix(int[][] matrix) {
    int m = matrix.length, n = matrix[0].length;
    sum = new int[m + 1][n + 1];
    for (int i = 1; i <= m; i++)
        for (int j = 1; j <= n; j++)
            sum[i][j] = matrix[i - 1][j - 1] + sum[i][j - 1] + sum[i - 1][j] - sum[i - 1][j - 1];
}

public int sumRegion(int row1, int col1, int row2, int col2) {
    return sum[row2 + 1][col2 + 1] - sum[row2 + 1][col1] - sum[row1][col2 + 1] + sum[row1][col1];
}
preSum[i][j] = matrix[i-1][j-1] + sum[i][j-1] + sum[i-1][j] - sum[i-1][j-1]
