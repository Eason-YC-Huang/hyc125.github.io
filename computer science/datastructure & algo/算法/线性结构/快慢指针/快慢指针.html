825. 适龄的朋友
对于用户i，他会对年龄在区间(0.5*ages[i]+7,ages[i]]内的人发出好友申请 对ages进行排序,遍历每个用户的年龄,随着用户年龄的增加,他的目前区间也随其右移
public int numFriendRequests(int[] ages) {
    int len = ages.length;
    Arrays.sort(ages);
    int ans = 0;
    int lo = 0, hi = 0;
    for (int age : ages) {
        // 这行很重要0.0
        if (age <= 14) continue;
        while (ages[lo] <= 0.5 * age + 7) lo++;
        while (hi < len && ages[hi] <= age) hi++;
        ans += hi - lo - 1;
    }
    return ans;
}
11. 盛最多水的容器
左右指针
高度不变的情况下，左右指针的收缩会导致面积的减小，这是如果要让面积变大，则需要让当前较短的柱子变长，也就是，假设左指针的柱子低，则左指针右移，否则右指针左移
public int maxArea(int[] height) {
    int len = height.length;
    int ans = 0;
    int lo = 0, hi = len - 1;
    while (lo < hi) {
        int area = (hi - lo) * (Math.min(height[hi], height[lo]));
        ans = Math.max(ans, area);
        if (height[lo] < height[hi]) lo++;
        else hi--;
    }
    return ans;
}

15. 三数之和
数组排序 遍历数组 左右指针 对于第一个元素，如果等于前一个元素，跳过即可 对于左右指针对应的元素，如果等于下一个元素，下一个元素跳过
public List<List<Integer>> threeSum(int[] nums) {
    int len = nums.length;
    Arrays.sort(nums);
    List<List<Integer>> ans = new ArrayList<>();
    for (int i = 0; i < len - 2; i++) {
        // if (nums[i] > 0) break; 不建议写优化，不然四数和容易翻车
        if (i > 0 && nums[i] == nums[i - 1]) continue;
        int lo = i + 1, hi = len - 1;
        while (lo < hi) {
            int sum = nums[i] + nums[lo] + nums[hi];
            if (sum > 0) while (hi > lo && nums[hi] == nums[--hi]) ;
            else if (sum < 0) while (lo < hi && nums[lo] == nums[++lo]) ;
            else {
                ans.add(Arrays.asList(nums[i], nums[lo++], nums[hi--]));
                while (hi > lo && nums[hi] == nums[--hi]) ;
                while (lo < hi && nums[lo] == nums[++lo]) ;
            }
        }
    }
    return ans;
}
611. 有效三角形的个数
首先对数组排序，然后固定最长的一条边，运用双指针扫描，如果 nums[l]+nums[r]>nums[i]nums[l] + nums[r] > nums[i]nums[l]+nums[r]>nums[i]，则有nums[l+1]+nums[r]>nums[i],...,nums[r−1]+nums[r]>nums[i]nums[l + 1] + nums[r] > nums[i], ..., nums[r - 1] + nums[r] > nums[i]nums[l+1]+nums[r]>nums[i],...,nums[r−1]+nums[r]>nums[i]，满足的条件的有 r - l 种，r 左移进入下一轮。
如果 nums[l] + nums[r] <= nums[i]，l 右移进入下一轮。
枚举结束后，总和就是答案。
时间复杂度为 O(n2)O(n^2)O(n2)
public int triangleNumber(int[] nums) {
    Arrays.sort(nums);
    int n = nums.length;
    int res = 0;
    for (int i = n - 1; i >= 2; --i) {
        int l = 0, r = i - 1;
        while (l < r) {
            if (nums[l] + nums[r] > nums[i]) {
                res += r - l;
                --r;
            } else ++l;
        }
    }
    return res;
}
18. 四数之和
public List<List<Integer>> fourSum(int[] nums, int target) {
    int len = nums.length;
    Arrays.sort(nums);
    List<List<Integer>> ans = new ArrayList<>();
    for (int i = 0; i < len - 3; i++) {
        // 优化
        // 如果这四数之和大于target, 往后算也必大于target
        if ((long) nums[i] + nums[i + 1] + nums[i + 2] + nums[i + 3] > target) break;
        // 下面这四数之和小于target, 则nums[i]小了
        if ((long) nums[i] + nums[len-3] + nums[len-2] + nums[len-1] < target) continue;
        // 跟三数和不同，这里target有可能是负数
        // 考虑nums=[-5,-4,-1] target=-10
        // 所以这里不能用 if (a > target) break;
        if (i > 0 && nums[i] == nums[i - 1]) continue;
        for (int j = i + 1; j < len - 2; j++) {
            if (j > i + 1 && nums[j] == nums[j - 1]) continue;
            int lo = j + 1, hi = len - 1;
            while (lo < hi) {
                int sum = nums[i] + nums[j] + nums[lo] + nums[hi];
                if (sum > target) while (hi > lo && nums[hi] == nums[--hi]) ;
                else if (sum < target) while (lo < hi && nums[lo] == nums[++lo]) ;
                else {
                    ans.add(Arrays.asList(nums[i], nums[j], nums[lo], nums[hi]));
                    while (hi > lo && nums[hi] == nums[--hi]) ;
                    while (lo < hi && nums[lo] == nums[++lo]) ;
                }
            }
        }
    }
    return ans;
}

public int threeSumClosest(int[] nums, int target) {
    int len = nums.length;
    Arrays.sort(nums);
    int ans = nums[0] + nums[1] + nums[2];
    for (int i = 0; i < len; i++) {
        if (i > 0 && nums[i] == nums[i - 1]) continue;
        int lo = i + 1, hi = len - 1;
        while (lo < hi) {
            int sum = nums[i] + nums[lo] + nums[hi];
            if (sum > target) while (hi > lo && nums[hi] == nums[--hi]) ;
            else if (sum < target) while (lo < hi && nums[lo] == nums[++lo]) ;
            else return sum;
            if (Math.abs(sum - target) < Math.abs(ans - target)) ans = sum;
        }
    }
    return ans;
}
1995. 统计特殊四元组
/**
 * nums[a] + nums[b] + nums[c] = nums[d] ->
 * nums[a] + nums[b] = nums[d] - nums[c]
 * 逆序遍历b
 *    统计目前所有的d与新出现的c的差值 -> count(d-c)
 *    遍历目前所有的a, ans+=count(a+b)
 * 由于 nums[d]−nums[c] 可能为负，在使用数组代替哈希表时，可利用 1<=nums[i]<=100 做一个值偏移。
 *
 */
public int countQuadruplets(int[] nums) {
    int n = nums.length, ans = 0;
    int[] cnt = new int[401];
    for (int b = n - 3; b >= 1; b--) {
        int c = b + 1;
        for (int d = b + 2; d < n; d++) cnt[nums[d] - nums[c] + 200]++;
        for (int a = 0; a < b; a++) ans += cnt[nums[a] + nums[b] + 200];
    }
    return ans;
}
19. 删除链表的倒数第 N 个结点
快慢指针 快指针先走n步，然后快慢指针同时前进 当快指针走到最后一个元素时，慢指针达到了待删除元素的前面
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode();
    dummy.next = head;
    ListNode lo = dummy, hi = dummy;
    for (int i = 0; i < n; i++) hi = hi.next;
    while(hi.next != null){
        hi = hi.next;
        lo = lo.next;
    }
    // 其实这里应该还要把lo.next.next设为null，不然lo.next无法被GC
    lo.next = lo.next.next;
    return dummy.next;
}

public void merge(int[] nums, int m, int[] arr, int n) {
  int len = nums.length;
  int i = m - 1, j = n - 1;
  for (int k = len - 1; k >= 0; k--) {

345. 反转字符串中的元音字母
static boolean[] hash = new boolean[128];
static char[] vowels = new char[]{'a', 'e', 'i', 'o', 'u'};

static {
    for (char c : vowels) {
        hash[c - ' '] = hash[Character.toUpperCase(c) - ' '] = true;
    }
}

public String reverseVowels(String s) {
    char[] cs = s.toCharArray();
    int len = s.length();
    int lo = 0, hi = len - 1;
    while (lo < hi) {
        while (!hash[cs[lo] - ' ']) lo++;
        while (!hash[cs[hi] - ' ']) hi--;
        if (lo >= hi) break;
        swap(cs, lo++, hi--);
    }
    return String.valueOf(cs);
}

void swap(char[] cs, int l, int r) {
    char c = cs[l];
    cs[l] = cs[r];
    cs[r] = c;
}
581. 最短无序连续子数组
整个数组可以分成如下图所示的三段
考虑中段数组，设其左边界为L，右边界为R：
很明显：[L,R]中的最大值等于[0,R]中的最大值(max[L,R]==max[0,R]max[L,R]==max[0,R]max[L,R]==max[0,R]) ,
同时：nums[R] 不可能是[0，R]中的最大值(我们设其为 max)（否则 nums[R] 应该算进右端数组）
由此可见：nums[R] < max < nums[R+1] < nums[R+2] < ...
也就是说：右边界R是从左往右遍历最后一个小于当前最大值的元素
同样原理：[L,R]中的最小值等于[L,len-1]中的最小值(min[L,R]==min[L,len−1]min[L,R]==min[L,len-1]min[L,R]==min[L,len−1])
同时：nums[L]不可能是[L,len-1]中的最小值(我们设其为min)
由此可见：... < nums[L-1] < nums[L-2] < min < nums[L]
也就是说：左边界L是从右往左遍历最后一个大于当前最小值的元素
public int findUnsortedSubarray(int[] nums) {
    int len = nums.length;
    if (len == 1) return 0;

    int hi = 0, max = nums[0];
    for (int i = 0; i < len; i++) {
        if (nums[i] < max) hi = i;
        else max = nums[i];
    }

    int lo = len - 1, min = nums[len - 1];
    for (int i = len - 1; i >= 0; i--) {
        if (nums[i] > min) lo = i;
        else min = nums[i];
    }
    // 如果数组本身有序,hi和lo是不会改变的,依然是0和len-1
    return hi == 0 ? 0 : hi - lo + 1;
}

public boolean judgeSquareSum(int c) {

1764. 通过连接另一个数组的子数组得到一个数组
public boolean canChoose(int[][] gs, int[] nums) {
    int m = gs.length, n = nums.length;
    int i = 0, j = 0;
    while (i < m && j < n) {
        if (check(gs[i], nums, j)) {
            j += gs[i].length;
            i++;
        } else {
            j++;
        }
    }
    return i == m;
}

// 从nums的idx开始,能否找到跟target一样的子数组
private boolean check(int[] target, int[] nums, int idx) {
    if (target.length > nums.length - idx) return false;

    for (int i = 0; i < target.length; i++, idx++)
        if (target[i] != nums[idx]) return false;

    return true;
}

930. 和相同的二元子数组
三指针滑动窗口 右指针扩展，左指针找到第一个符合要求的位置 中间指针从左指针位置开始，找到第一个不符合要求的位置 中间指针到左指针的距离就是该窗口的结果
public int numSubarraysWithSum(int[] nums, int t) {
    int len = nums.length;
    int ans = 0;
    int cnt1 = 0, cnt2 = 0;
    for (int lo = 0, mid = 0, hi = 0; hi < len; hi++) {
        cnt1 += nums[hi];
        cnt2 += nums[hi];

        while (cnt1 > t) cnt1 -= nums[lo++];
        // 左指针必然是不会超过hi指针的
        // 但是中间指针是有可能会超过hi的，所以要做限制
        // 考略[0,0,0] 0 中间指针不停右移都无法满足条件，最终会溢出
        while (mid <= hi && cnt2 >= t) cnt2 -= nums[mid++];
        ans += mid - lo;
    }
    return ans;
}
