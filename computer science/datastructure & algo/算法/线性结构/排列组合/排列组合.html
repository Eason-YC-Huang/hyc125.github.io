排列组合

排列就是排队，顺序很重要，组合就是组队，人一样就行了
排列英文名叫 Arrangement 或者 Permutation，组合英文名叫 Combination
n个物品排列的结果
n! = n*(n-1)*(n-2)...1
n个物品取k个，要注意顺序
n!/((n-k)!)
n个物品取k个，顺序不重要
n!/((n-k)!)/k!
有nnn个数，每个数有xxx种状态，则一共有xnx^{n}xn种状态组合
什么时候使用 used 数组，什么时候使用 start 变量？
排列问题，讲究顺序，数组指针可以往前移动，所以需要用 used 数组记录哪些数字已经使用过。
for(int i=0;i<n;i++){
  if(used[i])continue;
  used[i]=true;
  dfs(nums)
  used[i]=false;
}
组合问题，不讲究顺序，数组指针只能往后移动，此时使用 start 变量记录数组的开始位置。
// 从start开始,对于每一个数,我可以选或者不选
// 当i=start表示我选start
// 当i=start+1表示我没有选start
for(int i=start;i<n;i++){
  if(used[i])continue;
  used[i]=true;
  dfs(nums,i+1);
  used[i]=false;
}
无论是排列还是组合，核心都是这个数，我可以选，也可以不选。 但是对于排列，可以先跳过这个数，后面再选。 但是组合不可以，对于组合来说，过了这个村就没这个店了。
46. 全排列
private List<List<Integer>> ans = new ArrayList<>();
private List<Integer> sub = new ArrayList<>();
private boolean[] used;

public List<List<Integer>> permute(int[] nums) {
    used = new boolean[nums.length];
    dfs(nums);
    return ans;
}

private void dfs(int[] nums) {
    if (sub.size() == nums.length) {
        ans.add(new ArrayList<>(sub));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        sub.add(nums[i]);
        used[i] = true;
        dfs(nums);
        used[i] = false;
        sub.remove(sub.size() - 1);
    }
}
47. 全排列 II (去重)
List<List<Integer>> ans = new ArrayList<>();
List<Integer> sub = new ArrayList<>();
boolean[] used;

public List<List<Integer>> permuteUnique(int[] nums) {
    Arrays.sort(nums);
    used = new boolean[nums.length];
    dfs(nums);
    return ans;
}

public void dfs(int[] nums) {
    if (sub.size() == nums.length) {
        ans.add(new ArrayList<>(sub));
        return;
    }
    for (int i = 0; i < nums.length; ++i) {
        if (used[i]) continue;
        // 去重的核心
        if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) continue;

        sub.add(nums[i]);
        used[i] = true;
        dfs(nums);
        used[i] = false;
        sub.remove(sub.size() - 1);
    }
}
526. 优美的排列
int ans = 0;
boolean[] used;

public int countArrangement(int n) {
    used = new boolean[n + 1];
    dfs(n, 1);
    return ans;
}

// 如果不需要用辅助列表sub里面的元素,只需要sub的长度,我们可以用一个cnt变量来记录
private void dfs(int n, int cnt) {
    if (cnt > n) {
        ans++;
        return;
    }
    for (int num = 1; num <= n; num++) {
        if (used[num]) continue;
        if (cnt % num == 0 || num % cnt == 0) {
            used[num] = true;
            dfs(n, cnt + 1);
            used[num] = false;
        }
    }
}
667. 优美的排列 II(这题跟排列组合一点关系都没有.....)
整个结果数组可以看做两部分：
  • 第一部分有k个数，它们产生k-1个不同相邻差
  • 第二部分有n-k个数，它们是第k个数的顺序递增或递减。
如果理解不了上面三句话，没问题，从这里开始看就可以了。
对于序列[1,n] 和 k，只需要从序列的首尾交替取k个数，然后按照最后一个数递增或递减。
例如n=9，k=5，我们只需要从序列的首尾交替取5个数即可。
nums: 1,9,2,8,3
diff:  8,7,6,5
现在已经有4个不同的相邻差，接下来的数自然递增可以了。
nums: 1,9,2,8,3,4,5,6,7
diff:  8 7 6 5 1 1 1 1
一共有且仅有5个不同的相邻差。
public int[] constructArray(int n, int k) {
    int[] ans = new int[n];
    int lo=1, hi=n;
    // 交替选前k个数
    for(int i=0;i<k;i++) ans[i] = i%2==0? lo++ : hi--;
    // 处理剩余的数(如果k是偶数，剩余的数要从hi开始递减，否则从lo开始递增)
    for(int i=k;i<n;i++) ans[i] = k%2==0? hi-- : lo++;
    return ans;
}
时间复杂度：O(N)
空间复杂度：O(1)
77. 组合 (典型组合：过了这村没这个店，一个店只能进一次)
List<List<Integer>> ans = new ArrayList<>();
List<Integer> sub = new ArrayList<>();

public List<List<Integer>> combine(int n, int k) {
    int[] nums = IntStream.rangeClosed(1, n).toArray();
    dfs(nums, 0, k);
    return ans;
}

private void dfs(int[] nums, int start, int k) {
    if (sub.size() == k) {
        ans.add(new ArrayList<>(sub));
        return;
    }

    for (int i = start; i < nums.length; i++) {
        sub.add(nums[i]);
        dfs(nums, i + 1, k);
        sub.remove(sub.size() - 1);
    }
}

// 对于哪些范围是[1,n]的问题
// 你不要管那么多，反正start是从0开始的，他表示数组从哪里开始选
// nums[i]=i+1
List<List<Integer>> ans = new ArrayList<>();
List<Integer> sub = new ArrayList<>();

public List<List<Integer>> combine(int n, int k) {
    dfs(n, 0, k);
    return ans;
}

private void dfs(int n, int start, int k) {
    if (sub.size() == k) {
        ans.add(new ArrayList<>(sub));
        return;
    }

    for (int i = start; i < n; i++) {
        sub.add(i + 1);
        dfs(n, i + 1, k);
        sub.remove(sub.size() - 1);
    }
}
39. 组合总和 (过了这个村没这个店，这个店可以重复进)
List<List<Integer>> ans = new ArrayList<>();
List<Integer> sub = new ArrayList<>();

public List<List<Integer>> combinationSum(int[] candidates, int target) {
    Arrays.sort(candidates);
    dfs(candidates, target, 0);
    return ans;
}

private void dfs(int[] nums, int target, int idx) {
    if (target == 0) {
        ans.add(new ArrayList<>(sub));
        return;
    }

    for (int i = idx; i < nums.length; i++) {
        if (nums[i] > target) return;
        sub.add(nums[i]);
        dfs(nums, target - nums[i], i);
        sub.remove(sub.size() - 1);
    }
}

216. 组合总和 III
List<List<Integer>> ans = new ArrayList<>();
List<Integer> sub = new ArrayList<>();

public List<List<Integer>> combinationSum3(int k, int n) {
    dfs(0,k,n);
    return ans;
}

private void dfs(int start, int k, int sum){
    if(sub.size()==k && sum==0){
        ans.add(new ArrayList<>(sub));
        return;
    }

    if(sub.size()==k) return;

    for(int i=start;i<9;i++){
        if(sum<i+1) return;
        sub.add(i+1);
        dfs(i+1,k,sum-i-1);
        sub.remove(sub.size()-1);
    }

}
17. 电话号码的字母组合
static String[] dict = {"abc", "def", "ghi", "jkl", "mno", "pqrs", "tuv", "wxyz"};

List<String> ans = new ArrayList<>();
StringBuilder sb = new StringBuilder();

public List<String> letterCombinations(String digits) {
    dfs(digits, 0);
    return ans;
}

private void dfs(String digits, int idx) {
    if (idx == digits.length()) {
        ans.add(sb.toString());
        return;
    }

    if (digits.charAt(idx) == '1') dfs(digits, idx + 1);
    String options = dict[digits.charAt(idx) - '2'];
    for (int i = 0; i < options.length(); i++) {
        sb.append(options.charAt(i));
        dfs(digits, idx + 1);
        sb.deleteCharAt(sb.length() - 1);
    }
}
