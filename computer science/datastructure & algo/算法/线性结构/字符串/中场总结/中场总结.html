中场总结

补充空元素

当发现dp[i-1][j-1] 时，可以考虑在最前面补充空元素，从而简化边界条件的判断
例如最长公共子序列问题
dp[i][j]：text1[0,i] 与 text2[0,j]的最长公共子序列
状态转移方程：dp[i][j] = text1[i] == text2[j] ? dp[i-1][j-1]+1 : max(dp[i-1][j],dp[i][j-1])
上面的状态定义，我们需要先处理i=0 和 j=0 时的边界问题
  public int longestCommonSubsequence(String text1, String text2) {
      int M = text1.length() , N = text2.length();
      // dp[i][j] text1[0,i] text2[0,j]的最长公共子序列
      int[][] dp = new int[M][N];

      // 初始化
      dp[0][0] = text1.charAt(0) == text2.charAt(0) ? 1 : 0 ;
      for(int i=1;i<M;++i){
          dp[i][0] = text1.charAt(i)==text2.charAt(0) ? 1:dp[i-1][0];
      }
      for(int j=1;j<N;++j){
          dp[0][j] = text1.charAt(0)==text2.charAt(j) ? 1:dp[0][j-1];
      }

      for(int i=1;i<M;++i){
          for(int j=1;j<N;++j){
              dp[i][j] = text1.charAt(i)==text2.charAt(j) ? dp[i-1][j-1]+1 : Math.max(dp[i][j-1],dp[i-1][j]);
          }
      }

      return dp[M-1][N-1];
  }
我们可以在两个字符串前面补充空串，从而避免初始化
  public int longestCommonSubsequence(String text1, String text2) {
      int M = text1.length() , N = text2.length();
      // dp[i][j] text1[0,i-1] text2[0,j-1]的最长公共子序列
      int[][] dp = new int[M+1][N+1];

      for(int i=1;i<M;++i){
          for(int j=1;j<N;++j){
              dp[i][j] = text1.charAt(i-1)==text2.charAt(j-1) ? 
                dp[i-1][j-1]+1 : Math.max(dp[i][j-1],dp[i-1][j]);
          }
      }
      
      return dp[M][N];
  }

优化成一维数组

还是最长公共子序列，可看出 dp[i][j]只与dp[i-1][j-1]、dp[i][j-1]、dp[i-1][j]有关，也就是只跟上一行和当前行有关系，因此我们可以把二维数组缩减成两个一维数组
  public int longestCommonSubsequence(String text1, String text2) {
      int M = text1.length() , N = text2.length();
      // 上一行的结果
      int[] dp = new int[N+1];

      for(int i=1;i<=M;++i){
          // 当前行的结果
          int[] subDp = new int[N+1];
          for(int j=1;j<=N;++j){
              subDp[j] = text1.charAt(i-1)==text2.charAt(j-1) ? dp[j-1]+1 : Math.max(subDp[j-1],dp[j]);
          }
          dp = subDp;
      }
      return dp[N];
  }
