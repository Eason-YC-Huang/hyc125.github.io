技巧：递增长度
给你一个数组nums,要求返回一个数组ans，ans[i]表示从nums的索引i开始往后计算，它的非严格递增长度是多少？ 例如 nums:[1,2,2,3,2] → ans:[3,2,1,0,0]。
ans[i] = nums[i]<nums[i+1] ? ans[i+1]+1 : 0;
int[] nums = {1, 2, 2, 3, 2};
int len = nums.length;
int[] ans = new int[len];
for (int i = len - 2; i >= 0; i--) {
    ans[i] = nums[i + 1] >= nums[i] ? ans[i + 1] + 1 : 0;
}
技巧：后前两数差值符合要求的二元组
前缀和常常跟一个技巧一起使用，这个技巧我叫作，后前两数差值符合要求的二元组
给定数组nums，找出二元组[i,j]使得nums[j]-nuns[i]符合某个条件，且0≤i<j<n
例如给定nums，统计nums[j]-nums[i]=goal的二元组的个数。
因为nums[j]-nums[i]=goal所以nums[j]-goal=nums[i]
我们可以边遍历边记录每个元素的出现次数，
在处理nums[j]的时，找nums[j]-goal的出现次数
private int count(int[] nums, int goal) {
    int ans = 0;
    Map<Integer, Integer> cnt = new HashMap<>();
    for (int num : nums) {
        int target = num - goal;
        ans += cnt.getOrDefault(target, 0);
        cnt.put(num, cnt.getOrDefault(num, 0) + 1);
    }
    return ans;
}
