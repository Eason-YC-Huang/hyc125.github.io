300. 最长递增子序列
方法1
dp[i] ：从0 ~ i 且最后一个数字是nums[i]时，最长子序列长度
求解 dp[i] 时，向前遍历找出比 i 元素小的元素 j，令 dp[i] 为 max(dp[i],dp[j]+1)
最后获取 dp 列表最大值，即可得到全局最长上升子序列长度
方法2
使用一条单调递增的队列，当新元素进队时，如果没有其它元素比它大，新元素放到末尾，否则，用新元素代替第一个大于等于它的元素 (最长递增队列)
等价于：二分查找，找出第一个大于等于目标值的位置，如果没有，返回最新的位置
public int lengthOfLIS(int[] nums) {
    // 最长递增队列:新元素入队的位置,第一个大于等于该元素的位置
    int[] queue = new int[nums.length];
    int ans = 0;
    for (int num : nums) {
        int lo = 0 , hi = ans;
        while (lo < hi) {
            int mid = (lo + hi) >> 1;
            if (queue[mid]>=num) hi = mid;
            else lo = mid + 1;
        }
        queue[hi] = num;
        if (hi == ans) ans++;
    }
    return ans;
}
时间复杂度 O（NLogN)O（NLogN)O（NLogN)
空间复杂度 O（N)O（N)O（N)

334. 递增的三元子序列
public boolean increasingTriplet(int[] nums) {
    // 求最长递增子序列是否大于等于3
    // 使用一条单调队列
    // 遍历nums,假设当前为x,用x替换队列中第一个大于等于它的元素
    // 其实它的本质是贪心,用更小的元素去替换第一个比它大的元素,那么就能增大后面的元素进入队列的机会
    int[] q = new int[3];
    int idx = 0;
    for (int num : nums) {
        int lo = 0, hi = idx;
        while (lo < hi) {
            int mid = (hi + lo) >> 1;
            if (q[mid] >= num) hi = mid;
            else lo = mid + 1;
        }
        q[lo] = num;
        if (lo == idx) idx++;
        if (idx == 3) return true;
    }
    return false;
}

1143. 最长公共子序列
dp[i][j] : text1[0:i] 和 text2[0:j] 的最长公共子序列
dp[i][j] = text1[i] == text2[j] ? dp[i-1][j-1]+1 : max(dp[i-1][j],dp[i][j-1])
  public int longestCommonSubsequence(String text1, String text2) {
      int M = text1.length() , N = text2.length();
      // dp[i][j] text1[0,i] text2[0,j]的最长公共子序列
      int[][] dp = new int[M][N];

      // 初始化
      dp[0][0] = text1.charAt(0) == text2.charAt(0) ? 1 : 0 ;
      for(int i=1;i<M;++i){
          dp[i][0] = text1.charAt(i)==text2.charAt(0) ? 1:dp[i-1][0];
      }
      for(int j=1;j<N;++j){
          dp[0][j] = text1.charAt(0)==text2.charAt(j) ? 1:dp[0][j-1];
      }

      for(int i=1;i<M;++i){
          for(int j=1;j<N;++j){
              dp[i][j] = text1.charAt(i)==text2.charAt(j) ? dp[i-1][j-1]+1 : Math.max(dp[i][j-1],dp[i-1][j]);
          }
      }

      return dp[M-1][N-1];
  }

1035. 不相交的线
其实就是求最长公共子序列
public int maxUncrossedLines(int[] nums1, int[] nums2) {
    // 其实就是求nums1和nums2的最长公共子序列
    int m = nums1.length , n = nums2.length;
    int[] dp = new int[n+1];
    for(int i=1;i<=m;i++){
        int[] cur = new int[n+1];
        for(int j=1;j<=n;j++){
            cur[j] = nums1[i-1]==nums2[j-1] ? dp[j-1]+1
                   : Math.max(cur[j-1],dp[j]);
        }
        dp = cur;
    }
    return dp[n];
}
128. 最长连续序列
入参其实就是由多组连续序列组成的数组
找出每个连续序列的第一个元素，然后统计该序列的长度
  1. 遍历所有元素，存入Set
  2. 遍历Set的元素，如果当前元素X减1不存在Set里面
  3. 则X是一个连续序列的第一个元素，然后就可以开始统计该序列的长度
public int longestConsecutive(int[] nums) {
    int ans = 0;
    Set<Integer> set = new HashSet<>();
    for (int num : nums) set.add(num);
    for (Integer num : set) {
        if (set.contains(num - 1)) continue;
        int len = 1;
        while (set.contains(num + len)) len++;
        ans = Math.max(ans, len);
    }
    return ans;
}
594. 最长和谐子序列
方法1. 排序然后滑动窗口 nlog(n)nlog(n)nlog(n) 方法2. 遍历数组，假设当前元素是xxx，那么xxx的和谐子数组的长度=xxx个数加上(((x−1x-1x−1)))的个数
public int findLHS(int[] nums) {
    int ans = 0;
    Map<Integer, Integer> cnt = new HashMap<>();
    for (int num : nums) cnt.put(num, cnt.getOrDefault(num, 0) + 1);
    for (int num : nums) {
        if (cnt.containsKey(num - 1)) {
            ans = Math.max(ans, cnt.get(num) + cnt.get(num - 1));
        }
    }
    return ans;
}

public String findLongestWord(String s, List<String> dictionary) {
    // 对列表进行排序，长度最长且字典序最小的字符串
    // 判断字符串是否是s的子序列
    List<String> dict = dictionary.stream()
                                  .filter(str->str.length()<=s.length())
                                  .sorted((x,y)->x.length()==y.length()? x.compareTo(y):y.length()-x.length())
                                  .collect(Collectors.toList());

    for(String p:dict){

    }

    return "";

}

public boolean isSubSeq(String p , String str){
    int M = p.length() , N = str.length();
    int i = 0 , j = 0;
    for(;i<N && j<M;){
        if(p.charAt(j)==str.charAt(i)){
            j++;
        }
        i++;
    }
    return j==M;
}

78. 子序列
集合的每个元素，都有可以选或不选，用二进制和位运算，可以很好的表示。
第iii位表示第iii个数的选中状态，000为不选，111为选
public static List<List<Integer>> binaryBit(int[] nums) {
    int len = nums.length;
    List<List<Integer>> ans = new ArrayList<>();
    for (int i = 0; i < (1 << len); i++) {
        List<Integer> sub = new ArrayList<>();
        for (int j = 0; j < len; j++)
            if (((i >> j) & 1) == 1) sub.add(nums[j]);
        ans.add(sub);
    }
    return ans;
}

大体跟上一题类似，但是需要考虑如何过滤重复
如果满足三个条件，则当前的选择状态为重复
1、当前元素被选中。2、当前元素等于前一个元素。3、前一个元素未选中。
j > 0 && nums[j] == nums[j - 1] && ((i >> (j - 1)) & 1) == 0
public List<List<Integer>> subsetsWithDup(int[] nums) {
    // 为什么要先排序？
    Arrays.sort(nums);
    int len = nums.length;
    List<List<Integer>> ans = new ArrayList<>();
    for (int i = 0; i < (1 << len); i++) {
        List<Integer> sub = new ArrayList<>();
        for (int j = 0; j < len; j++)
            if (((i >> j) & 1) == 1) {
                if (j > 0 && nums[j] == nums[j - 1] && ((i >> (j - 1)) & 1) == 0) {
                    sub = null;
                    break;
                }
                sub.add(nums[j]);
            }
        if (sub != null) ans.add(sub);
    }
    return ans;
}
