背包问题的本质

有 n 种物品和容量为capacity的背包，第 i 件物品的体积为 v[i]，价值为 w[i]，你要选取物品放入到背包中。

背包的分类

常见的背包类型主要有以下几种： 1、0/1背包问题：每个元素最多选取一次 2、完全背包问题：每个元素可以重复选择 3、组合背包问题：背包中的物品要考虑顺序 4、分组背包问题：不止一个背包，需要遍历每个背包

最值问题

问题：n个物品中选取，刚好装满背包，能得到的最大价值
dp[i][j] 前i个物品中选取，体积恰好为j时，最大的价值
如果第i个物品不取，则dp[i][j]=dp[i-1][j]
如果第i个物品取，则dp[i][j]=dp[i-1][j-v[i]]+w[i]
dp[i][j] = max(dp[i-1][j],dp[i-1][j-v[i]]+w[i])
public int zeroOnePackage(int[] w, int[] v, int capacity) {
  int quantity = w.length;
  int[][] dp = new int[quantity][capacity + 1];
  // dp[i][j] = 0~i的物品中选取,容量刚好为j的最大价值
  // dp[i][j] = max(dp[i-1][j],dp[i-i][j-v[i]]+w[i])

  // 初始化
  dp[0][v[0]] = w[0];

  for (int i = 1; i < quantity; ++i) {
    for (int j = capacity; j>=0; --j) {
      dp[i][j] = dp[i - 1][j];
      if (j >= v[i]) {
        dp[i][j] = Math.max(dp[i][j], dp[i - 1][j - v[i]] + w[i]);
      }
    }
  }

  return dp[quantity - 1][capacity];
}

public int zeroOnePackage(int[] w, int[] v, int capacity) {
  int quantity = w.length;
  int[][] dp = new int[quantity][capacity + 1];
  // dp[i][j] = 0~i的物品中选取,容量不超过j的最大价值
  // dp[i][j] = max(dp[i-1][j],dp[i-i][j-v[i]]+w[i])

  // 初始化 dp[0][j] = j>=v[0] ? w[0] : 0
  for(int j=capacity;j>=v[0];--j){
    dp[0][j] = w[0];
  }

  for (int i = 1; i < quantity; ++i) {
    for (int j = capacity; j>=0; --j) {
      if (j >= v[i]) {
        dp[i][j] = Math.max(dp[i - 1][j], dp[i - 1][j - v[i]] + w[i]);
      }else{
        dp[i][j] = dp[i - 1][j];
      }
    }
  }

  return dp[quantity - 1][capacity];
}

最值问题

问题：n个物品中选取，不超过背包容量，能得到的最大价值
dp[i][j] 前i个物品中选取，体积恰好为j时，最大的价值
如果第i个物品不取，则dp[i][j]=dp[i-1][j]
如果第i个物品取1个，则dp[i-1][j-v[i]]+w[i]
如果第i个物品取2个，则dp[i-1][j-2*v[i]]+2*w[i]
....
如果第i个物品取k个，dp[i-1][j-k*v[i]]+k*w[i]

dp[i][j] = max(dp[i-1][j],dp[i-1][j-v[1]]+w[1],..,dp[i-1][j-k*v[i]]+k*w[i])
int[][] dp = new int[n][capacity+1];

// 初始化
for(int i=0;i<=capacity;++i){
  int maxK = i/v[0];
  dp[0][i] = maxK*w[0];
}

for(i=1;i<n;++i){
  for(j=0;j<=capacity;++j){
    dp[i][j] = dp[i-1][j];
    int y = 0;
    for(int k=1;k<=j/v[i];++k){
      y = max(y,dp[i-1][j-k*v[i]]+k*w[i])
    }
    dp[i][j] = max(dp[i][j],y)
  }
}

总结

首先是背包分类的模板：
  1. 0/1背包：外循环nums,内循环target,target倒序且target>=nums[i];
  2. 完全背包：外循环nums,内循环target,target正序且target>=nums[i];
  3. 组合背包：外循环target,内循环nums,target倒序且target>=nums[i];
  4. 分组背包：这个比较特殊，需要三重循环：外循环背包bags,内部两层循环根据题目的要求转化为1,2,3三种背包类型的模板
然后是问题分类的模板：
  1. 最值问题
    dp[i] = max/min(dp[i], dp[i-nums]+1)
或
dp[i] = max/min(dp[i], dp[i-num]+nums);
  2. 存在问题(bool)
    dp[i]=dp[i]||dp[i-num];
  3. 方案个数问题
    dp[i]+=dp[i-num];
