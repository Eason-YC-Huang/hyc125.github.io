846. 一手顺子
public boolean isNStraightHand(int[] hand, int groupSize) {
    // 数组长度要能被size整除
    // 数组从小到大排序
    // 用hash表记录每个元素的剩余个数
    // 遍历数组，如果当前元素有剩余元素，以该元素为首，尝试能否组成符合要求的顺子
    // 如果不能返回结果false
    // 如果能，对应的元素的个数要减一
    if (hand.length % groupSize != 0) return false;
    Arrays.sort(hand);
    Map<Integer, Integer> cnt = new HashMap<>();
    for (int num : hand)
        cnt.put(num, cnt.getOrDefault(num, 0) + 1);

    for (int num : hand) {
        if (cnt.get(num) == 0) continue;
        for (int i = 0; i < groupSize; i++) {
            Integer total = cnt.getOrDefault(num + i, 0);
            if (total == 0) return false;
            cnt.put(num + i, total - 1);
        }
    }
    return true;
}
5980. 将字符串拆分为若干长度为 k 的组
计算分组数，填充字符，然后分组
public String[] divideString(String s, int k, char fill) {
    int len = s.length();
    int size = (int) Math.ceil((double) len / k);
    for (int i = 0; i < size * k - len; i++) s += fill;

    String[] ans = new String[size];
    int lo = 0, hi = k;
    for (int i = 0; i < size; i++) {
        ans[i] = s.substring(lo, hi);
        lo = hi;
        hi += k;
    }
    return ans;
}
区间分组：符合某一特征且元素相邻的为一组
public void partition(int[] nums) {
    int len = nums.length;
    for (int lo = 0, hi = 0; hi < len; hi++, lo = hi) {
        while (check(nums, hi)) hi++;
        // 走到这里的hi是该区间最后一个符合要求的元素
        operation(nums, lo, hi);
    }
}
5958. 股票平滑下跌阶段的数目
设某平滑下降区间的元素个数是nnn，则该区间的平滑下降阶段的数目等于 n∗(n+1)2\frac{n*(n+1)}{2}2n∗(n+1)​(其实就是子数组个数)
累加所有区间的平滑下降阶段的数目即为结果
注意结果返回的是一个long
public long getDescentPeriods(int[] nums) {
    int len = nums.length;
    int lo = 0, hi = 0;
    long ans = 0L;
    while (hi < len) {
        while (check(nums, hi)) hi++;
        ans += operation(nums, lo, hi);
        hi++; lo = hi;
    }
    return ans;
}

private boolean check(int[] nums, int hi) {
    if (hi+1>=nums.length) return false;
    return nums[hi] == nums[hi+1] + 1;
}

private long operation(int[] nums, int lo, int hi) {
    // 这里有个坑，很容易写成
    // return (hi-lo+1)*(hi-lo+1+1)/2;
    // 因为已经溢出了，即使转为long结果也是错的
    long n = hi-lo+1;
    return n*(n+1)/2;
}
443. 压缩字符串
区间分组+写指针,确定好区间后, 把当前区间的字符写入写指针指向的位置 计算当前区间的长度,写入写指针指向的位置
public int compress(char[] chars) {
    int len = chars.length;
    int ans = 0, writeIdx = 0;
    for (int lo = 0, hi = 0; hi < len; hi++, lo = hi) {
        while (hi + 1 < len && chars[hi] == chars[hi + 1]) hi++;
        ans++;
        chars[writeIdx++] = chars[hi];

        int size = hi - lo + 1;
        if (size > 1) {
            String num = "" + size;
            for (int i = 0; i < num.length(); i++) {
                chars[writeIdx++] = num.charAt(i);
                ans++;
            }
        }
    }
    return ans;
}
485. 最大连续 1 的个数
区间分组，返回最长的区间长度
public int findMaxConsecutiveOnes(int[] nums) {
    int len = nums.length;
    int ans = 0;
    for (int lo = 0, hi = 0; hi < len; hi++, lo = hi) {
        if (nums[hi] == 0) continue;
        while (hi + 1 < len && nums[hi] == nums[hi + 1]) hi++;
        ans = Math.max(ans, hi - lo + 1);
    }
    return ans;
}
