647.回文子串个数统计

647题是找到一个字符串中所有的回文子串，而第 5 题是求解一个字符串中最长的回文子串，很明显求解出所有的字符串自然能够找到最大的，所以我们先看 647题。 1. 动态规划法
状态：dp[i][j] 表示字符串s在[i,j]区间的子串是否是一个回文串。
状态转移方程：
dp[i][j] = s[i] == s[j] && (j - i < 2 || dp[i + 1][j - 1]) ? true : false
这个状态转移方程是什么意思呢？
  • 当只有一个字符时，比如 a 自然是一个回文串。
  • 当有两个字符时，如果是相等的，比如 aa，也是一个回文串。
  • 当有三个及以上字符时，比如 ababa 这个字符记作串 1，把两边的 a 去掉，也就是 bab 记作串 2，可以看出只要串2是一个回文串，那么左右各多了一个 a 的串 1 必定也是回文串。所以当 s[i]==s[j] 时，自然要看 dp[i+1][j-1] 是不是一个回文串。
所以最后代码为：
    public int countSubstrings(String s) {
        // dp[i][j] 索引i~j是否是回文串
        // dp[i][j] = s[i]==s[j] && (j-i<2||dp[i+1][j-1])

        int ans = 0;
        int len = s.length();
        boolean[][] dp = new boolean[len][len];

        for (int i = len - 1; i >= 0; --i) {
            for (int j = i; j < len; ++j) {
                if (dp[i][j] = Objects.equals(s.charAt(i), s.charAt(j)) && (j - i < 2 || dp[i + 1][j - 1])) ans++;
            }
        }

        return ans;
    }

5.最长回文子串的长度

遍历每个字符，从当前字符C开始左右扩展，先扩展到s[lo]≠C && s[hi]≠C，然后比较s[lo]和s[hi]，如果相同，则继续扩展lo和hi
  public String longestPalindrome(String s) {
      int len = s.length();
      if (len < 2) {
          return s;
      }

      int curLen = 0;
      int maxLen = 1;
      int maxLenStart = 0;
      for (int i = 1; i < len; ++i) {
          char c = s.charAt(i);
          int lo = i - 1, hi = i + 1;
          while (lo >= 0 && s.charAt(lo) == c) lo--;
          while (hi < len && s.charAt(hi) == c) hi++;
          while (lo >= 0 && hi < len && s.charAt(lo) == s.charAt(hi)) {
              lo--;
              hi++;
          }
          curLen = hi - lo - 1;
          if (maxLen < curLen) {
              maxLen = curLen;
              maxLenStart = lo + 1;
          }
      }

      return s.substring(maxLenStart, maxLenStart + maxLen);
  }

516.最长回文子序列的长度

1. 动态规划法
状态：dp[i][j] 表示字符串s在[i,j]区间的最长回文子串的长度
状态转移方程：
dp[i][j] = s[i] == s[j] ? dp[i+1][[j-1]+2 : max(dp[i+1][j],dp[i,j-1])
  public int longestPalindromeSubseq(String s) {
      int len = s.length();
      int[][] dp = new int[len][len];
      for (int i = len - 1; i >= 0; i--) {
          dp[i][i] = 1;
          for (int j = i + 1; j < len; j++) {
              dp[i][j] = s.charAt(i) == s.charAt(j) ? dp[i + 1][j - 1] + 2 
              : Math.max(dp[i + 1][j], dp[i][j - 1]);
          }
      }
      return dp[0][len - 1];
  }

1332. 删除回文子序列
如果s本身是回文串，只需删除一次即可。否则最大的删除次数为 2（先删除所有的 a，再删除所有的 b）。
public int removePalindromeSub(String s) {
    int len = s.length();
    int lo = 0, hi = len - 1;
    while (lo < hi)
        if (s.charAt(lo++) != s.charAt(hi--)) return 2;

    return 1;
}
