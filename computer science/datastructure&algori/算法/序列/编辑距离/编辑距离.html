392.判断子序列

双指针
本题询问的是，ppp是否是 strstrstr的子序列，因此只要能找到任意一种 ppp 在 strstrstr 中出现的方式，即可认为 ppp 是 strstrstr 的子序列。
而当我们从前往后匹配，可以发现每次贪心地匹配靠前的字符是最优决策。
假设当前需要匹配字符ccc，而字符cc c在strstrstr中的位置x1x1x1 和x2 x2 x2出现（x1<x2x1<x2x1<x2），那么贪心取x1x1x1是最优解，因为x2x2x2后面取到的字符，x1x1x1也能取到，并且通过x1x1x1与x2x2x2之间的可选字符，更有希望匹配成功
我们初始化两个指针 iii 和 jjj，分别指向 ppp 和 strstrstr 的初始位置。指针对应的两个元素做匹配，如果匹配成功则 iii 和 jjj 同时右移，匹配 ppp 的下一个位置，如果匹配失败则 jjj 右移，iii 不变，尝试用 strstrstr 的下一个字符匹配 ppp。最终如果 iii 移动到 pp p 的末尾，就说明 pp p 是 strstrstr 的子序列。
  public boolean isSubsequence(String p, String str) {
      int n = p.length(), m = str.length();
      int i = 0, j = 0;
      while (i < n && j < m) {
          if (p.charAt(i) == str.charAt(j)) {
              i++;
          }
          j++;
      }
      return i == n;
  }
复杂度分析
时间复杂度：O(n+m)O(n+m)O(n+m)，其中 nnn 为 sss 的长度，mmm 为 ttt 的长度。每次无论是匹配成功还是失败，都有至少一个指针发生右移，两指针能够位移的总距离为 n+mn+mn+m。
空间复杂度：O(1)O(1)O(1)。

  public int numDistinct(String str, String p) {
      // dp[i][j] p[0,i]在str[0,j]的子序列中出现的次数
      // dp[i][j] = p[i] == str[j] ? dp[i-1][j-1] + dp[i][j-1] : dp[i][j-1]
      // 看到dp[i-1][j-1] 我们填充一个0，减少边界条件的判断
      // dp[i][j] p的前i个字符 在 str的前j个字符 的子序列中出现的次数
      // dp[0][0] 表示 p 和 str 都是空串  dp[0][0] = 1
      // dp[1][0] 表示 str 是空串 dp[1][0] = 0
      // dp[0][1] 表示 p 是空串 dp[0][1] = 1

      int M = p.length() , N = str.length();
      int[][] dp = new int[M+1][N+1];
      for(int j=0;j<=N;++j) dp[0][j] = 1;
      for(int i=1;i<=M;++i){
          for(int j=1;j<=N;++j){
              dp[i][j] = p.charAt(i-1)==str.charAt(j-1) ? (dp[i-1][j-1]+dp[i][j-1]) : dp[i][j-1];
          }
      }
      return dp[M][N];
  }
