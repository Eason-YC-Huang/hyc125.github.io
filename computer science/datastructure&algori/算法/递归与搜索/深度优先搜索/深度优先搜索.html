网格的DFS
网格结构要比二叉树结构稍微复杂一些，它其实是一种简化版的图结构。要写好网格上的 DFS 遍历，我们首先要理解二叉树上的 DFS 遍历方法，再类比写出网格结构上的 DFS 遍历。我们写的二叉树 DFS 遍历一般是这样的：
void traverse(TreeNode root) {
    // 递归终止条件
    if (root == null) {
        return;
    }
    // 访问两个相邻结点：左子结点、右子结点
    traverse(root.left);
    traverse(root.right);
}
可以看到，二叉树的 DFS 有两个要素：「访问相邻结点」和「递归终止条件」。
第一个要素是访问相邻结点。二叉树的相邻结点非常简单，只有左子结点和右子结点两个。二叉树本身就是一个递归定义的结构：一棵二叉树，它的左子树和右子树也是一棵二叉树。那么我们的 DFS 遍历只需要递归调用左子树和右子树即可。
第二个要素是递归终止条件。一般来说，二叉树遍历的递归终止条件是 root == null。这样一个条件判断其实有两个含义：一方面，这表示 root 指向的子树为空，不需要再往下遍历了。另一方面，在 root == null 的时候及时返回，可以让后面的 root.left 和 root.right 操作不会出现空指针异常。
对于网格上的 DFS，我们完全可以参考二叉树的 DFS，写出网格 DFS 的要素：
  1. 网格结构中的格子有多少相邻结点？答案是上下左右四个。对于格子 (i, j) 来说，四个相邻的格子分别是 (i-1, j)、(i+1, j)、(i, j-1)、(i, j+1)。
  2. 网格 DFS 中的递归终止条件是什么？从二叉树的递归终止条件对应过来，应该是网格中不需要继续遍历、grid[i][j] 会出现数组下标越界异常的格子，也就是那些超出网格范围的格子。
  3. 网格DFS不同于树的DFS，网格DFS在遍历中可能遇到遍历过的结点。这是因为，网格结构本质上是一个「图」，我们可以把每个格子看成图中的结点，每个结点有向上下左右的四条边。在图中遍历时，自然可能遇到重复遍历结点。所以我们需要使用一个集合来记录已经遍历过的结点。
这样，我们得到了网格 DFS 遍历的框架代码：
void dfs(int[][] grid, int i, int j , Set<String> seen) {
    
    
    // 递归终止条件，数组越界
    if (i < 0 || j < 0 || i >= grid.length || j >= grid[0].length) return;
    
    // 递归终止条件，重复访问
    String key = i + "-" + j;
    if (seen.contains(key)) return;

    // 记录已经访问的元素
    seen.add(key);
    
    // 访问上、下、左、右四个相邻结点
    dfs(grid, i - 1, j, seen);
    dfs(grid, i + 1, j, seen);
    dfs(grid, i, j - 1, seen);
    dfs(grid, i, j + 1, seen);
}

boolean outOfArea(int[][] grid, int i, int j) {
    return i < 0
            || j < 0
            || i >= grid.length
            || j >= grid[0].length;
}

472. 连接词
1.字符串按照长度从短到长排序，确保当前处理的字符，其组合字符已经记录在前缀树
2.当前字符匹配到一个组合字符时，有两种情况，开始新字符的匹配，继续匹配
private static class Node {
    boolean isWord = false;
    Map<Character, Node> child = new HashMap<>();
}

private Node dummy = new Node();

public void add(String str) {
    Node cur = dummy;
    for (int i = 0; i < str.length(); i++)
        cur = cur.child.computeIfAbsent(str.charAt(i), x -> new Node());
    cur.isWord = true;
}

public boolean contains(String str) {
    Node cur = dummy;
    for (int i = 0; i < str.length(); i++) {
        Node child = cur.child.get(str.charAt(i));
        if (child == null) return false;
        cur = child;
    }
    return cur.isWord;
}

public List<String> findAllConcatenatedWordsInADict(String[] words) {
    // 看到题目第一反应就是前缀树,但是在思考[cat,cats,dog,catsdog]尬住了
    // ⭐️⭐️⭐️当你遇到这样可能可以,那样也可能可以的情况时,要考虑下dfs⭐️⭐️⭐️
    List<String> ans = new ArrayList<>();

    Arrays.sort(words, Comparator.comparingInt(String::length));
    for (String word : words) {
        if (word.length() == 0) continue;
        if (dfs(word, 0)) ans.add(word);
        else add(word);
    }

    return ans;
}

// str[idx:]是否是连接词
private boolean dfs(String str, int idx) {
    if (idx == str.length()) return true;
    Node cur = dummy;
    for (int i = idx; i < str.length(); i++) {
        Node child = cur.child.get(str.charAt(i));
        if (child == null) return false;
        if (child.isWord && dfs(str, i + 1)) return true;
        else cur = child;
    }
    return false;
}
